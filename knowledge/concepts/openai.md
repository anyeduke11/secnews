# OpenAI

## 定义

美国 AI 研究公司，开发了 GPT 系列大语言模型、DALL-E 图像生成模型和 ChatGPT 产品。是生成式 AI 领域的领军企业。

## 关键要点

1. GPT 系列模型推动 LLM 规模化应用
2. API 经济的先行者，催生大量 AI 应用
3. 商业化路径：订阅 + API + 企业版
