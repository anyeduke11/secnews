## Graph JSON Schema

`knowledge/concepts/graph.json` — knowledge graph snapshot, Agent-written, backend-cached.

```json
{
  "nodes": [
    { "id": "zero-trust", "label": "零信任", "type": "concept", "wiki": "hotspot", "count": 23 },
    { "id": "xdr", "label": "XDR", "type": "concept", "wiki": "hotspot", "count": 12 },
    { "id": "local:zero-trust", "label": "零信任", "type": "concept", "wiki": "local", "count": 8 }
  ],
  "edges": [
    { "source": "zero-trust", "target": "xdr", "type": "related", "weight": 5 },
    { "source": "zero-trust", "target": "local:zero-trust", "type": "federated", "weight": 1 }
```

Node fields: `id` (slug), `label` (display title), `type` (`concept` | `item`), `wiki` (`hotspot` | `local`), `count` (source_items length).
Edge fields: `source`, `target` (node ids), `type` (`related` | `federated`), `weight` (co-occurrence count).

## Content Calendar JSON Schema

`knowledge/content/calendar.json` — publishing calendar, date-keyed.

```json
{
  "2026-07-20": {
    "topic": "零信任架构趋势",
    "type": "深度分析",
    "status": "planned",
    "source_items": ["a1b2c3", "d4e5f6"],
    "draft_path": "knowledge/content/drafts/zero-trust-trend.md",
    "platform": "wechat",
    "published_url": null,
    "stats": null
  }
}
```

Top-level: object keyed by `YYYY-MM-DD` date string.
Per-date fields:
- `topic` (required, string)
- `type` (string, e.g. 深度分析 / 快讯 / 事件分析)
- `status` (`planned` | `drafting` | `published`)
- `source_items` (array of item ids)
- `draft_path` (relative path to draft .md, nullable)
- `platform` (`wechat` | `x` | `weibo` | `xhs`)
- `published_url` (string, nullable)
- `stats` (object with `views`/`likes` integers, nullable)
