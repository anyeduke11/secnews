# 一款完全离线的自主渗透测试智能体AIRecon

AIRecon 是一款完全离线运行的自主渗透测试智能体，它将自托管的 Ollama 大语言模型与 Kali Linux Docker 沙箱相结合，能够自动化完成端到端的安全评估，且全程不向云端暴露任何数据。
